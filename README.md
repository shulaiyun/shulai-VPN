# 树懒VPN（SlothVPN）

> 基于 Hiddify 深度定制的多平台代理客户端，主打美观、易用与高度可定制化。

[GitHub 仓库](https://github.com/shulaiyun/shulai-VPN) | [Telegram 频道](https://t.me/shulai2026)

## 项目简介

树懒VPN（SlothVPN）是一个面向多平台分发和品牌化二开的代理客户端项目。

项目基于 Hiddify 深度定制打造，重点不在“简单换皮”，而在于对整体体验、品牌能力、构建链路和多平台交付进行持续整理，使其更适合独立品牌运营、定制分发和后续功能扩展。

## 项目亮点

- 现代化界面：在保留核心能力的前提下，持续优化 UI、交互和视觉表现。
- 品牌化能力：支持独立名称、图标、链接、更新源和分发渠道，适合品牌化二开。
- 多平台支持：项目面向 Android、Windows、Linux、macOS、iOS 工程统一维护。
- 可持续构建与发布：仓库已接入 GitHub Actions，逐步完善自动构建、产物归档和 Release 分发。

## 当前定位

- 项目名称：树懒VPN（SlothVPN）
- 项目方向：美观、品牌化、可定制化、多平台分发
- 适用场景：自有品牌客户端、定制发行、多平台统一维护

## 平台支持

当前仓库包含以下平台工程：

- Android
- Windows
- Linux
- macOS
- iOS

其中：

- Android 与 Windows 已优先整理为可持续构建的主线平台。
- Linux 与 macOS 已纳入仓库与工作流规划，后续继续完善云端构建稳定性与签名分发。
- iOS 工程保留，用于后续接入 macOS + Xcode 环境下的签名与分发流程。

## 关于上游项目

本项目基于 [Hiddify](https://github.com/hiddify-com/hiddify-next) 深度定制打造。

上游项目提供了重要的跨平台代理客户端基础能力，本项目在此基础上进行了品牌、构建、分发、平台适配和体验层面的定制化整理。树懒VPN并不否认其上游来源，但仓库文案、品牌展示和后续分发将以“树懒VPN”作为主品牌进行维护。
